Rudiments d'informatique

Pour que l'on parle tous le même langage

Un algorithme

Ensemble d'opérations élémentaires dans le but de résoudre un problème.

  • Pas forcément informatique
  • Pas forcément implémenté

Un programme (informatique)

L'implémentation informatique d'un algorithme

  • Plusieurs type (batch, interactif)
  • Compilé/interprété
  • mono/multi langae
  • Dissociation programme/donnée

La mentalité d'un informaticien

Analyser un problème complexe d'apparence insoluble et le découper en taches simples faciles à résoudre puis assembler le tout. Moins je code mieux je me porte

Les variables : opérateur numérique

  • + - * / : opérateur de base
  • ** : puissance
  • // : division euclidienne
  • % : modulo

Les variables : opérateur booléen

  • == : égalité
  • < <= > >
  • != : différent
  • and; or; not
